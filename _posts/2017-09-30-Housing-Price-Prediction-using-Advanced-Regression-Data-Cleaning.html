Housing Price Prediction using Advanced Regression¶

The Business Problem: The dataset of the project is of residential homes in Ames, Iowa. It contains 79 different house attributes/parameters. The end goal of the project is to apply Feature Engineering and Advanced Regression Techniques like Random Forest or Gradient Boosting to predict the final price of the house using the given list of variables.

The dataset for can be found here.

This project had all the aspects of typical data science challenges ranging from Data Cleaning & Transformation to Feature Engineering and finally Modelling.

Since all these aspects had equal weightage in the project. I divided my Project Analysis into 3 separate Notebooks.

  1. Data Cleaning
  2. Feature Engineering
  3. Modelling

with open('data_description.txt','r') as file:
    content = file.readlines()
    for line in content:
        line.rstrip('\n')
content.insert(-1,':')


#Create a function to get just the information from data_description file

def info(data_list,x,data):
    for a in range(len(data_list)):
        if (x+':') in data_list[a]:
            for b in range(a+1,len(data_list)):
                if ':' in data_list[b]:
                    for c in range(a,b):
                        print(data_list[c])
                    break
                else:
                    continue
        else:
            continue

# Define a fucntion to fill the missing values
def fill_missing(data,column, value):
    data.loc[data[column].isnull(),column] = value
